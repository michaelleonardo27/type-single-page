History of the font

In OpenType's frontier, DJR pioneers thrive, Variations unite, fonts' diversity in one archive. Pixel mastery defines LAB's innovative drive, Each glyph a testament, meticulously alive. From Los Angeles to Hampshire, skills they derive, A foundry's legacy, with Emily, they strive.

LAB DJR's variable font offers designers a range of customizable attributes to finely tune typographic elements. From adjusting weight for hierarchy and emphasis to modifying width for optimal text layout, designers can also adapt optical size for legibility across different sizes and control italic angle for stylistic diversity. Contrast variations influence texture and visual impact, while grade adjustments allow for different visual impacts in display typefaces. X-height modifications affect legibility and overall texture, while terminal style changes add subtle visual interest. These features ensure optimal aesthetics, readability, and expressive versatility across diverse design applications.
